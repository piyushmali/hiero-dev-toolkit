# Contributing

Thanks for contributing to `@hiero-dev/toolkit`.

## Development

```bash
npm install
npm run typecheck
npm run build
npm test
```

## Pull Request Checklist

1. Add tests for behavior changes.
2. Keep public APIs documented with JSDoc.
3. Run `npm run typecheck && npm run build && npm test`.
4. Update README/examples for DX-facing changes.

## DCO Requirement

This project requires Developer Certificate of Origin (DCO) sign-off on every commit.

Use:

```bash
git commit -s -m "feat: add mirror client enhancement"
```

This adds `Signed-off-by: Your Name <you@example.com>` to the commit message.
